# 🧠 Math Quiz Game (C++)

## 📖 Description
This project is a **console-based Math Quiz Game** written in C++.  
The program generates random math questions and allows the player to solve them.  
At the end of the quiz, the program evaluates the results and shows the final score.

This project is part of my **early programming practice** while learning **Computer Science and Computer Engineering fundamentals**.

---

## 🎯 Features

- Generate **random math questions**
- Multiple **difficulty levels**
- Different **operation types**
- Tracks:
  - Correct answers
  - Wrong answers
- Shows **final quiz results**
- Provides **visual feedback** using console colors
- Option to **play again**

## 🎮 How the Game Works

1. The player selects:
   - Number of questions
   - Difficulty level
   - Operation type

2. The program generates random questions based on the selected settings.

3. The player answers each question.

4. The program checks the answers and gives immediate feedback.

5. At the end, the game displays:
   - Total questions
   - Number of correct answers
   - Number of wrong answers
   - Final result (Pass / Fail)
